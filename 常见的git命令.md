# 命令

## 本地的操作
1. git init
2. git add ./
3. git commit

## 分支操作
git branch
git checkout (branch)

## 查看状态
git satus 

## 设置远程仓库
git remote add origin (remote store adreess)
git remote rm origin

## 拉取文件
git pull origin (branch)
git pull --rebase origin (branch)

## 上传文件
git push origin (branch)

## 克隆文件
git clone (address)
